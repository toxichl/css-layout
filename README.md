# CSS layout

Well-designed CSS layout samples

## Layout List

- Two Columns Layout
	- via `float`
	- via `negative margin`
	- via `flexbox`
	
- Three Columns Layout
	- via `negative margin`
	- via `flexbox`

## Suggestion

- Two columns in PC-side —— `float`;
- Three columns in PC-side —— `margin`;
- Mobile application —— `flex`。
